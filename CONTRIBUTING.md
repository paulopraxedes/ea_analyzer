# Como Contribuir

Obrigado por considerar contribuir com o EA Analyzer Web.

## Requisitos

- Python 3.10+
- Node.js 18+
- MetaTrader 5 instalado para testes locais do backend

## Fluxo de Trabalho

1. Faça um fork do repositório
2. Crie um branch a partir de `main`
3. Faça alterações pequenas e objetivas
4. Atualize ou adicione testes quando aplicável
5. Rode `npm run lint` e `npm run build` no frontend
6. Abra o pull request com descrição clara

## Padrões de Código

- Mantenha o estilo existente do projeto
- Evite alterações não relacionadas ao objetivo do PR
- Prefira utilitários já existentes no código

## Reportando Problemas

Inclua informações completas:

- Passos para reproduzir
- Resultado esperado e atual
- Logs relevantes
- Versões de Node, Python e MT5
